## 📂 Proyectos Destacados

### 1. [NutriFit AI] — Planificación Nutricional Inteligente (Android Nativo)
*Aplicación móvil nativa para Android que estructura dietas y rutinas de ejercicio procesando lenguaje natural mediante Inteligencia Artificial.*
*   **Stack:** Kotlin, Jetpack Compose, Room DB, Arquitectura MVVM.
*   **Características clave:**
    *   Arquitectura robusta e intuitiva offline-first mediante Room DB.
    *   Integración directa de APIs de Inteligencia Artificial para el análisis inmediato de descripciones libres de alimentos.
    *   Diseño de interfaz declarativa fluida con transiciones dinámicas.

### 2. [Telegram App-Citas] — Automatización conversacional con Whisper (Node.js)
*Bot inteligente para la concertación de citas automáticas mediante voz, diseñado originalmente en el marco de EsarIA.*
*   **Stack:** Node.js, Telegram Bot API, API de Whisper (Local / Nube), REST APIs.
*   **Características clave:**
    *   Transcripción de notas de voz de clientes en tiempo real utilizando Whisper.
    *   Parseo inteligente de intenciones ("quiero una cita mañana a las 5") e integración directa con calendarios y bases de datos en la nube.
    *   Estructura modular para fácil portabilidad entre canales (Telegram/WhatsApp).
